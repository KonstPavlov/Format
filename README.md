# Сайт компании ФОРМАТ — ремонт квартир под ключ (Иркутск)

Премиальный лендинг: React (frontend) + FastAPI (backend). Форма заявки

## Портфолио: как добавить новый объект с локальными фотографиями

Все фотографии портфолио хранятся **локально** в проекте:

```
frontend/public/images/portfolio/
├── object-1/          # первый объект
│   ├── 1.jpg
│   ├── 2.jpg
│   └── 3.jpg
├── object-2/
│   ├── 1.jpg
│   └── ...
├── ...
└── placeholder.svg    # заглушка при отсутствии фото
```

Внешние ссылки (Unsplash/Pexels) больше не используются.

### Шаги для добавления нового объекта

1. **Создайте папку** для нового объекта:
   ```
   frontend/public/images/portfolio/object-7/
   ```

2. **Скопируйте фотографии** в эту папку и назовите их по порядку:
   ```
   1.jpg
   2.jpg
   3.jpg
   ```
   Фото может быть любое количество — просто продолжайте нумерацию.
   Рекомендуемый формат: `.jpg`, ширина ~1200 px (для быстрой загрузки).

3. **Добавьте объект в данные.** Откройте файл:
   ```
   frontend/src/constants/data.js
   ```
   Найдите массив `PORTFOLIO_ITEMS` и добавьте новый блок (можно скопировать
   любой существующий и поправить):
   ```js
   {
     id: 7,                                   // уникальный номер
     title: "Название проекта",               // заголовок карточки
     desc: "ЖК / улица — Площадь: 60 м²",      // подпись (локация, площадь)
     images: [
       "/images/portfolio/object-7/1.jpg",
       "/images/portfolio/object-7/2.jpg",
       "/images/portfolio/object-7/3.jpg"
     ],
     tags: ["Дизайнерский", "Гостиная"]        // теги (используются в фильтре)
   }
   ```

4. **Готово.** Галерея, слайдер (свайп/стрелки/клавиатура), лайтбокс и анимации
   работают автоматически — менять код не нужно.

### Как заменить фото у существующего объекта

Просто положите новый файл с тем же именем (например, `1.jpg`) в папку
соответствующего объекта. Путь в `data.js` менять не требуется.

### Фильтр по категориям

Кнопки-фильтры над галереей задаются в `frontend/src/App.js` в переменной
`allFilterTags`. Значения должны совпадать со значениями в `tags` объектов.

### Обработка отсутствующих фото

Если файл изображения не найден, вместо него автоматически показывается
заглушка `frontend/public/images/portfolio/placeholder.svg`
(через обработчик `onError` в `App.js`). Сайт не ломается.
